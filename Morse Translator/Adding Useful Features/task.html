Description

In the final stage, you will augment the application by adding a feature that provides users with a Morse code dictionary. This enhancement serves both as a learning tool and a quick reference, increasing the app's educational value. The Morse code dictionary should be seamlessly integrated into the existing user interface.

Objectives
  1. Create an AlphabetActivity that contains a RecyclerView with id rv_alphabet and a TextView tv_description
  2. RecyclerView item should contain tv_letter TextView, divider TextView with a "|" as a text, tv_morse TextView.
  3. RecyclerView should contain each letter of English alphabet with a corresponding morse code.
  4. Instantiate a SpannableStringBuilder to set the tv_description TextView's
    • use this text for description: "Letters are separated with one space, words are separated with 3 spaces . Unknown symbols will be displayed as '?'"
    • use spans to style with bold and italic formatting these pieces of the text 'one space' and '3 spaces'
